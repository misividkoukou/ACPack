A datapack for minecraft that tweaks some features in order to make the game better!
Made by AreoCraft Developers!

一个通过调整与增加一些小功能来提升服务器游戏体验的小数据包！

## 特性
* 计分板：包括死亡数、容器交互数、挖掘数、交易次数等，并支持轮播。
* 小提示：支持自定义的服务器内游玩提示系统，定时广播，支持独立开关。
* 扫地机：由玩家自助启停的掉落物清扫系统。
* 游玩教程：在新玩家首次加入时播报的游玩提示系统，支持修改。
* 配方修改：修改了部分原版游戏内的配方，增强生电服可玩性。
* 战利品表修改
* ...
